# Titanic Survival Prediction System
Developed by: Lambert Rhema (22CG031880)

## Setup Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Run training script: `python model/model_development.py`
3. Launch Web App: `python app.py`

## Features Used
- Pclass (1, 2, or 3)
- Sex (Male/Female)
- Age
- SibSp (Siblings/Spouses)
- Fare

## Input Range Examples
- Survived: Class 1, Female, Age 22, Fare 71.2
- Did Not Survive: Class 3, Male, Age 34, Fare 8.05
